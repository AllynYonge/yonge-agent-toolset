---
name: wireframe-before-major-ui
description: 重大系统页面设计前必须先用 ASCII 线框图说明方案并等待用户确认
type: feedback
---

在进行重大系统页面设计（包括但不限于页面重构、UI/UX 重设计、新页面架构）时，必须先输出 ASCII 线框图辅助说明设计方案，等待用户确认后再进入实施阶段。

**Why:** 页面级设计属于不可逆性较高的变更，用户需要在实施前直观理解布局和交互意图，避免返工。

**How to apply:** 任何涉及重大页面设计的任务（无论是否使用 impeccable 技能），第一步产出 ASCII 线框图 + 设计说明，暂停等待用户确认，确认通过后再生成代码。
