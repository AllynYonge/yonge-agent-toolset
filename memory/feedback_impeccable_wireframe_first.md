使用 `impeccable` 技能进行页面重构或 UI/UX 重新设计时，必须先输出 ASCII 线框图辅助说明设计方案，等待用户确认后再进入实施阶段。

**Why:** 页面重构属于不可逆性较高的变更，用户需要在实施前直观理解布局和交互意图，避免返工。

**How to apply:** 每次调用 impeccable 做页面级重构/重设计时，第一步产出 ASCII 线框图 + 设计说明，暂停等待用户确认，确认通过后再生成代码。
